Mantra Puja Release Notes (Version 4.0.0, Version Code 4):

- Devotional God Screen Enhancements:
  * Reordered pooja offerings options, placing the Aarti Thali first followed by Flower selection.
  * Added a direct-play toggle on the Aarti Thali button to start or stop the ritual instantly.
  * Synchronized the selected flower shower animation to fall and blossom continuously alongside the Aarti.
  * Integrated a premium Ujjain Mahakal Aarti background melody that loops automatically during the ritual.
  * Added periodic temple bell ringing and chime sound effects that trigger automatically every 3 seconds while Aarti is active.
  * Made the hanging bells highly responsive to manual taps.
- General performance improvements, stability updates, and API connectivity enhancements.
